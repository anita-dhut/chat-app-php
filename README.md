# 💬 Chat Application (ChitChat)

## 📌 Description
A web-based chat application developed using PHP and MySQL that allows users to register, log in, and communicate in real-time.

---

## 🚀 Features
- User registration and login system  
- Real-time chat functionality  
- Secure session handling  
- feedback forms  
- Logout functionality  

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: PHP  
- Database: MySQL  

---

## ⚙️ How to Run
1. Install XAMPP  
2. Place project in htdocs  
3. Import database from DB folder  
4. Run:
   http://localhost/chat-app
